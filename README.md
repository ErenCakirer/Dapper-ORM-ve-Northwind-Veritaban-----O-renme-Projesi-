# 🧩 Proje 5: Dapper ORM ve Northwind Veritabanı (Öğrenme Projesi)

Bu proje, **Microsoft’un Northwind veritabanı** ile **Dapper ORM** kullanarak temel veri işlemlerini gerçekleştirmek amacıyla hazırlanmıştır.  
Projenin odak noktası tasarım değil, **Dapper kullanımı**, **veritabanı bağlantısı** ve **CRUD mantığını öğrenmektir**.

---

## 🚀 Proje Amacı

Bu uygulamada Entity Framework yerine **Dapper ORM** kullanılarak SQL sorgularının daha hızlı ve sade bir şekilde nasıl yönetileceği gösterilmiştir.  
Proje **C# WinForms** arayüzüyle geliştirilmiş olup, kullanıcıların veritabanı ile etkileşim kurmasını sağlar.

---

## ✨ Özellikler

- **Northwind** veritabanına bağlantı sağlar  
- Temel **CRUD işlemleri** (Ekleme, Listeleme, Güncelleme, Silme) yapılabilir  
- **Dapper ORM** ile hafif ve hızlı veri erişimi  
- **DTO** ve **Repository** katman yapısı kullanılmıştır  
- Basit bir **WinForms** arayüzü içerir  

---

## 🛠️ Kullanılan Teknolojiler

- C# (.NET Framework / .NET 6)  
- Dapper ORM  
- WinForms  
- SQL Server (Northwind Veritabanı)  
- Repository & DTO yapısı  

---

## 📌 Notlar

> Bu proje **öğrenme amacıyla** hazırlanmıştır.  
> Veriler örnek olarak alınmış, arayüz tasarımı basit tutulmuştur.  
> Amaç, Dapper ORM kullanarak veritabanı işlemlerini kavramaktır.

---

## 🧠 Öğrenilen Konular

- **Dapper ORM** ile veritabanı işlemleri gerçekleştirme  
- **CRUD (Create, Read, Update, Delete)** işlemlerini uygulama  
- **Katmanlı mimari (Repository - DTO)** kullanımı  
- **WinForms** ile SQL Server arasında bağlantı kurma  

---

## 🪄 Çalıştırma Adımları

1. Bu projeyi **klonlayın** veya ZIP olarak indirin.  
2. **Visual Studio** içinde açın.  
3. NuGet üzerinden **Dapper** paketini yükleyin.  
4. `App.config` dosyasındaki bağlantı cümlesini (connection string) kendi veritabanınıza göre güncelleyin.  
5. Bilgisayarınızda **Northwind** veritabanının kurulu olduğundan emin olun.  
6. Projeyi çalıştırın ve CRUD işlemlerini deneyin.
